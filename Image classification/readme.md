# Rock-Paper-Scissors Image Classification using Tensorflow

### Overview
Course assignment for "Kelas Machine Learning untuk Pemula" held by Dicoding Academy. This machine learning model can recognize hand drawings that make up paper, scissors, or rock with an accuracy of 96%.

### Data Source
Dicoding resource
