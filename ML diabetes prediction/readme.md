# Diabetes Prediction using Machine Learning Random Forest Algorithm

### Overview
This is the source code that I use to predict whether someone has diabetes or not. The accuracy of this machine learning model is around 92% with an AUC of 0.94, this model is made using the Random Forest Classifier algorithm with several tuning parameters. This source code includes Exploratory Data Analysis in it.

### Data Source
https://www.kaggle.com/uciml/pima-indians-diabetes-database.html
