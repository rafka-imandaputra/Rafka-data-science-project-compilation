# Machine Learning Fraud Detection

### Overview
This is a machine learning with Python code that is used to predict whether someone is committing fraud or not based on 112 available features/column.

### Data Source
https://www.kaggle.com/datasets/volodymyrgavrysh/fraud-detection-bank-dataset-20k-records-binary
